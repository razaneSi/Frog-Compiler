# Frog-Compiler
this a third year compiler project we were asked to create a compiler/interpreter for the FROG programming language with a responsive interface .





avaiblable functionalities:

° testing with lexer


° verify the syntax


° verify the semantic 


° generation syntax tree



Results section shows :


° tokens reconized


° calculations


° input strings


Errors section shows:


° lexical errors


° syntax errors


° semantic errors


° an error a time( to avoid Cascading error)

Additional functionalities:


° Logs


° Documentation


° mini map 


° Light/ dark mode


for the GUI we used PyQt5
